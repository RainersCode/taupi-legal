# taupi-legal

Public legal pages for the [Taupi](https://github.com/RainersCode) mobile app, hosted on GitHub Pages.

## Live URLs

Once GitHub Pages is enabled on this repo (`Settings → Pages → Source: main branch → /(root)`), the pages are served at:

- `https://rainerscode.github.io/taupi-legal/` — landing page
- `https://rainerscode.github.io/taupi-legal/privacy-policy/`
- `https://rainerscode.github.io/taupi-legal/terms-of-service/`
- `https://rainerscode.github.io/taupi-legal/delete-account/`

These URLs are what Apple App Store Connect and Google Play Console need in their "Privacy Policy URL", "Support URL", and "Account Deletion URL" fields.

## Updating

Edit the markdown files on the `main` branch. GitHub Pages rebuilds automatically (usually within a minute). Bump the **Last updated** date and version number at the top of the file for every material change.
